# 🤖 AI Expense Analyzer

A smart expense tracking application that uses AI to categorize expenses and provide spending insights.

## Features

- 💰 Automatically categorizes expenses based on description
- 📊 Provides real-time spending insights
- 📈 Shows category-wise expense breakdown
- 💡 Offers personalized money-saving tips

## How to Use

1. Enter the expense amount
2. Add a description of the expense
3. Click "Add Expense"
4. Get instant categorization and insights!

## Categories

The AI recognizes several expense categories including:
- Food & Dining
- Transportation
- Entertainment
- And automatically categorizes other expenses based on amount

## Technologies Used

- Python
- Gradio
- AI-powered categorization logic
